# Risen 2 3D vision fix
========

This fix is currently work in progress.

#####What is fixed:
- Stars
- Water reflections
- Crosshair moved to depth
- HUD moved to depth
- Shadows
- Lights (torches on walls, still some glitches at some very narrow angles)

#####What is not fixed:
- Point lights, reflections (wet rocks, ...)
